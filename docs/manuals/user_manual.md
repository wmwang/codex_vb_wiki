# 使用者手冊（User Manual）

## 1. 目的
本手冊提供日常使用者（維運、開發、系統分析）如何使用本專案產出的 VB Deep Wiki 內容與 Speaker 規格輸出的指引。

## 2. 角色與權限
- **Viewer**：可瀏覽 wiki 與查詢。
- **Editor**：可更新內容（需審核）。
- **Reviewer**：審核內容與敏感資訊遮罩。
- **Owner**：模組負責人，對內容正確性負責。

## 3. 主要功能
- **Wiki 瀏覽**：以模組頁面、流程與依賴資訊查看系統內容。
- **Speaker 規格**：以 YAML/JSON 方式讀取結構化輸出。
- **搜尋/問答**：透過索引或 RAG 問答快速定位資訊（規劃導入）。

## 4. 快速開始
1. 開啟 `samples/photodemon/wiki/index.md` 查看模組索引。
2. 點選模組頁面（如 `PhotoDemonCore.md`）查看詳細內容。
3. 若需結構化資料，開啟 `samples/photodemon/speaker/*.yaml`。

## 5. 使用流程建議
- 需求分析階段：先看 **系統概覽** 與 **模組摘要**。
- 問題排查階段：確認 **流程** 與 **錯誤處理** 區段。
- 遷移/重構階段：以 **Speaker 規格** 為基礎建立 mapping。

## 6. 注意事項
- 任何內容變更需遵循審核流程（參考治理手冊）。
- 遇到敏感資訊（帳號、連線字串）請用遮罩。
